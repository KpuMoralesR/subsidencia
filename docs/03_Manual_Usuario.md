# 03. Manual de Usuario

## 1. Acceso al Sistema
Para ingresar, navegue a la página de inicio y haga clic en "Access Portal".

![Landing Page](./capturas/landpage_visitor.png)

Ingrese sus credenciales en el formulario de Login.

![Login Page](./capturas/login_usuario.png)

## 2. Dashboard Principal
Una vez autenticado, verá el Dashboard principal con el resumen de su cuenta.

![Dashboard](./capturas/dashboard_admin.png)

## 3. Gestión de Usuarios
En el menú lateral, seleccione "Users". Aquí puede visualizar, crear y editar usuarios.

![User Management](./capturas/modulo_usuarios_admin.png)

## 4. Gestión de Roles
En el menú lateral, seleccione "Roles". Defina qué módulos puede ver cada rol.

![Role Management](./capturas/modulo_roles_admin.png)
